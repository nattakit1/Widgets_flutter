# Flutter Navigation and Routing

## 1. Using the Navigator
# หลักการทำงาน

Navigator เป็นระบบจัดการหน้าจอแบบ Stack

การเปิดหน้าใหม่ใช้ push()

การย้อนกลับใช้ pop()

Flutter จะจัดเก็บหน้าจอไว้ใน Stack ดังนี้:

Screen1 → push → Screen2

Screen2 → pop → กลับ Screen1

# ลักษณะการทำงานในแอป

มีปุ่มสำหรับไปหน้าถัดไป

มีปุ่มย้อนกลับ

แสดงโค้ดตัวอย่างด้านล่างหน้าจอ
